# Jira Time Tracker

A terminal-based UI (TUI) application for logging time to Jira tickets, built with Rust.

## What is this?

A fast, keyboard-driven interface to track and log time spent on Jira issues without leaving your terminal.

## Features (WIP)

- View assigned Jira tickets
- Log time to issues
- Track time with built-in timer

## Tech Stack

- [Ratatui](https://github.com/ratatui-org/ratatui) - Terminal UI framework
- [Crossterm](https://github.com/crossterm-rs/crossterm) - Cross-platform terminal manipulation
- [Chrono](https://github.com/chronotope/chrono) - Date and time handling

## Getting Started

### Prerequisites

- Rust 1.70+ (install from [rustup.rs](https://rustup.rs))
- Jira account with API access

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/jira-time-tracker.git
cd jira-time-tracker

# Run the application
cargo run
```
